# 🚀 AI Workspace

A modern AI landing page built using React, TypeScript and Tailwind CSS.

---

## ✨ Features

- Modern Hero Section
- Floating Dashboard Cards
- Glassmorphism UI
- Responsive Layout
- Clean Component Structure
- Cloud Background Effects

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

---

## 📂 Project Structure

```
src/
 ├── assets/
 ├── components/
 │    ├── cards/
 │    ├── HeroSection.tsx
 │    ├── Navbar.tsx
 │    └── FloatingCards.tsx
 ├── App.tsx
 └── main.tsx
```

---

## ⚙️ Installation

```bash
git clone git@github.com:Gokztechz/AI-workspace.git

cd AI-workspace

npm install

npm run dev
```
